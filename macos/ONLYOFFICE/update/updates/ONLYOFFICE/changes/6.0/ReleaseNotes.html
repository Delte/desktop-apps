ONLYOFFICE 6.0 - October 15, 2020

Fixes

All Editors

  • Added the AutoFit settings for shapes.
  • Added the ability to insert special characters. Hotkeys have been implemented in the Document Editor only.
  • Added the Ctrl + 0 hotkey to reset zoom to 100%. A sequence fits a slide to workview in the Presentation Editor (Bug 45106).
  • Changed a behaviour of hiding icons in tabs when reducing editor windows. They are hiding as needed now.
  • Added the autorun settings for macros. If the Show Notification option is selected, a user will be notified about macros existence.
  • Added the ability to select an image from the document manager for watermarks, shapes and slides fill (Bug 42913).
  • Added the AutoCorrect list (Bug 44659) (DocumentServer issue #335).
  • Redesigned the color selection component.
  • Added the ability to change a position and size of chart elements (a title, an area, a legend and data labels).

Document Editor

  • Added the ability to insert date and time (Bug 36585).
  • Added the ability to print the selection in the view mode (Bug 44338).
  • Added the ability to convert equations from old formats (Bug 44511), (DocumentServer issue #780).
  • Changed a placeholder for content controls.
  • Redone the algorithm of justifying a paragraph with condensing spaces (Bug 40851, Bug 44960, DocumentServer issue #956).
  • Fixed an issue with an incorrect type of quotes for many localizations (Bug 45790, Bug 45591, DocumentServer issue #406, DocumentServer issue #898).
  • Fixed an issue with ignoring the menu key when calling the context menu (Bug 45692, DocumentServer issue #937).
  • Fixed an issue with replacing Chinese characters by digits in numbered lists (Bug 44723, DocumentServer issue #734).

Spreadsheet Editor

  • Added full support of pivot tables (inserting and editing).
  • Added the autofilter settings in pivot tables (by labels, by values, additional sort parameters).
  • Added support of opening all conditions existing in a file for data bar conditional formatting.
  • Added support of opening all gradients existing in a file for data bar conditional formatting.
  • Added the Shift+F3 hotkey for a function insertion.
  • Added full support of slicers for formatted tables (Bug 30397).
  • Added the special paste settings (Bug 42544).
  • Added the ability to move a sheet from one workbook to another.
  • Added the ability to insert an internal link to a named range (Bug 44821).
  • Added the ability to print titles (Bug 43869, DocumentServer issue #714).
  • Added a component for cells selection.
  • Added the Ctrl+Shift+= and Ctrl+Shift+- hotkeys for displaying the Insert Cells or Delete Cells dialogues (Bug 36285, DocumentServer issue #282).
  • Added the status bar settings.
  • Added the ability to remove duplicate values. Also available for formatted tables.
  • Added the Ctrl+; and Ctrl+Shift+; hotkeys for date and time insertion (Bug 42876).
  • Added the ability to insert function via the Function Arguments dialogue.
  • Added the ability to get a hyperlink to an internal range and open the workbook with linking to it.
  • Added the Wrap Text and Shrink to Fit settings in the right toolbar.
  • Added the Vertical Text option in the text orientation settings.
  • Combined the bulleted and numbered list settings. Added the ability to create lists from the List Settings dialogue.
  • Added the delimiter settings for the Special Paste and Text to Columns dialogues.
  • Added the ability to change the function in the Total Row for formatted tables.
  • Added the ability to select the column or row of a formatted table or the whole table by one mouse click.
  • Changed the cursor type for the column or row selection.
  • Added the LINEST function (Bug 45534).
  • Fixed an issue with calculating INDEX function if the row or column parameter is set to zero or omitted (Bug 45388, DocumentServer issue #955).
  • Fixed an issue with calculating FORMULATEXT function if it uses an array formula (Bug 45387, DocumentServer issue #391).
  • Fixed an issue with calculating DATE formula, if the month argument is set to zero or omitted (Bug 45533, DocumentServer issue #404).
  • Fixed an issue with calculating HYPERLINK function in French localization (Bug 45690, DocumentServer issue #913).
  • Fixed an issue with splitting pasted multiple lines of text to a few cells if it was copied from Microsoft Excel (Bug 15652, DocumentServer issue #750).
  • Fixed an issue with replacing data in cells by hash characters with vertical layout (Bug 42134, DocumentServer issue #618, DocumentServer issue #253).

Presentation Editor

  • Changed the interface for an internal link.
  • Added the ability to print the selection in the view mode (Bug 44338).
  • Combined the bulleted and numbered list settings. Added the ability to create lists from the List Settings dialogue.

and lots other fixes in all editors
